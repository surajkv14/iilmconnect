# **App Name**: IILM Connect

## Core Features:

- LMS Portal: Mimic the core functionalities of current LMS with live attendance tracking and notifications. Batch and classes will be handled in the admin module.
- CollabNest Portal: LinkedIn-like portal for students, faculty, and alumni. Features include profiles, posts, and interaction capabilities. University domain email verification required during sign-up.
- Mess Portal: Allows hostel students to pre-select food portions to minimize wastage. Mess staff can view selections to plan food preparation. University domain email verification required during sign-up.
- AI-Powered Mess Overview: Leverages historical consumption data and Gemini API for insights on student preferences. Staff inputs daily prepared/consumed quantities, analyzed to provide waste reduction overview tool.
- Admin Portal: Allows administrators to manage the LMS portal, assigning students to batches and teachers to classes to restrict access to relevant details.
- Email Verification: Require signup using a university domain email address to ensure that only IILM college students and staff can use the app.

## Style Guidelines:

- Primary color: Deep teal (#008080) for a professional and trustworthy feel.
- Background color: Light teal (#E0F8F8) for a clean and modern backdrop.
- Accent color: Light blue (#ADD8E6) to highlight interactive elements and calls to action.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and 'Inter' (sans-serif) for body text, ensuring a balance of modernity and readability.
- Use professional, clean icons that are easily understandable and consistent across the platform.
- A clean and structured layout to facilitate easy navigation and information access for all three portals.
- Subtle transitions and animations to enhance user experience without being distracting.
